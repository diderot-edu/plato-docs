Getting Started

Requesting an API Key​

Plato is in private beta right now, so you have to request an API key manually. Please contact: Andrew Gao, Roger Yang, or Umut Acar to obtain one. If you already have an API key, please go to the next step.

Installing Plato​

Plato requires Python >= 3.8, install using

pip install plato

Making your first query​

from Plato import Plato

client = Plato(<YOUR_API_KEY_HERE>)
client.upload(file="customer-support-docs.docx")
client.create("What's our return policy on items 30 days late?")
